# wiki-leyu-archive

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容来自不同的来源，以静态文件的形式保存，便于长期查阅与索引。

## 目录说明

```
.
├── index.html           # 仓库首页 / 归档入口
├── pages/               # 归档页面存放目录
│   ├── example-page.html
│   └── ...
├── assets/              # 共用资源文件（CSS、JS、图片等）
│   ├── style.css
│   └── ...
└── README.md            # 本文件
```

- `pages/`：存放所有归档的独立 HTML 页面，文件名通常反映页面主题或来源。
- `assets/`：存放页面引用的公共资源，如样式表、脚本和图像。

## 页面归档说明

- 每个 HTML 页面均为独立文件，不依赖外部数据库或后端服务。
- 页面内容在归档时尽量保持原始结构，仅做必要的本地化调整（如资源路径）。
- 归档不涉及对页面内容的评价、修改或补充说明。

## 维护说明

- 本仓库由维护者不定期更新，新增页面或修正链接。
- 如发现页面显示异常、资源缺失或信息过时，欢迎提交 Issue 或 Pull Request。
- 仓库内容仅供存档参考，不构成任何形式的推荐或保证。

## 贡献

欢迎通过以下方式参与：

- 提交 Issue 报告问题或提出建议
- 提交 Pull Request 修正内容或添加新页面
- 提供有价值的页面来源（请附上原始链接和简要说明）

## 许可

本仓库内容按各自页面原始版权状态处理。如有版权争议，请联系维护者处理。
